  Задание 3:
  
100 ВМ, Windows-based, нужны балансировщики, репликация, автоматический бэкап - VMware vSphere
Совместимость с Windows, встроенный балансировщик нагрузки и маршрутизация, наличие репликации и систем бэкапирования.
Бесплатное Open Source решение, 20–30 серверов, Linux + Windows ВМ, максимальная производительность - Proxmox VE (на базе KVM)
Proxmox полностью бесплатен, KVM самый производительный из Open Source решений. Proxmox из коробки предоставляет веб-интерфейс, кластеризацию, миграцию и встроенный механизм бэкапов. Современные версии Proxmox отлично работают с Windows Server.
Необходимо бесплатное, максимально совместимое и производительное решение для виртуализации Windows-инфраструктуры - Microsoft Hyper-V Server
Гипервизор от Microsoft, т.е. лучшая совместимость с Windows. Hyper-V Server — это бесплатный продукт Microsoft. Включает кластеризацию и репликацию в бесплатной версии.
Рабочее окружение для тестирования ПО на нескольких дистрибутивах Linuх -  VirtualBox
VirtualBox работает на рабочей станции инженера. Подходит для локального тестирования. Легко создать снапшот, изменить конфигурацию, разрушить окружение. С использованием Vagrant позволяет развернуть любую чиступю линукс машину за несколько минут.
